# 🛠️ SwiftCampFix

SwiftCampFix is a live college infrastructure complaint management system built to digitize and streamline campus issue reporting and resolution tracking.

🌐 **Live Application:**  
👉 https://swiftcampfix.free.nf/SwiftCampFix/frontend/index.php

---

## 🎯 Problem

In many colleges, infrastructure complaints (broken benches, water issues, electricity faults) are:

- Reported informally  
- Poorly tracked  
- Delayed in resolution  
- Lacking transparency  

---

## 💡 Solution

SwiftCampFix provides:

- 📝 Complaint submission system  
- 📊 Admin dashboard  
- 🔄 Status tracking (Pending / In Progress / Resolved)  
- 📂 Issue categorization  
- 📍 Location-based tagging  
- 📤 Complaint image upload
